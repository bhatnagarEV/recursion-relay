Recursion Relay

Start at the Start node. Solve the recursion question on the screen, then click the node whose displayed answer matches your result. Each correct click opens the next question. One wrong click on a question means that question earns 0 points, which makes random guessing a bad idea.

How scoring works
2 points for getting a question right on the first node click.
0 points for that question after any wrong node click.
You still must find the correct node to continue the trail.
